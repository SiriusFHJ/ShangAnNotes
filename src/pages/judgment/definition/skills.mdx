## 1. 关键词法
方法一：关键词——主体、客体
- 主体——行为、活动的发出者
- 客体——行为、活动作用的对象

**例题 1**
（2017 年江西）劳动争议是指在劳动者和劳动力使用者之间因劳动权利与义务发生分歧而引起的争议。根据上述定义，下列属于劳动争议的是：         
A．职工小周因工作调动的问题与当地劳动部门发生争执           
B．某企业职工小张和小李因工作上意见不合而产生矛盾           
C．两个单位之间因职工小刘的借调问题而产生矛盾           
D．职工老王因工伤未能获得保险赔偿而与工厂争执           


**解析**         
<BlurredAnswer>       
根据劳动争议的定义，即在劳动者和劳动力使用者之间因劳动权利与义务发生分歧而引起的争议，我们来分析一下每个选项：

A．职工小周因工作调动的问题与当地劳动部门发生争执 —— 这里的争议对象是“当地劳动部门”，而不是直接的劳动力使用者（即用人单位），不符合定义。

B．某企业职工小张和小李因工作上意见不合而产生矛盾 —— 这属于同事之间的矛盾，不是劳动者与用人单位之间的争议，不符合定义。

C．两个单位之间因职工小刘的借调问题而产生矛盾 —— 这里争议的主体是两个单位，不是劳动者与用人单位之间的争议，不符合定义。

D．职工老王因工伤未能获得保险赔偿而与工厂争执 —— 这涉及的是劳动者老王与他的用人单位（工厂）之间关于劳动权利（工伤赔偿）的争议，符合劳动争议的定义。

因此，**D**选项是属于劳动争议的。
</BlurredAnswer>

**例题 2**
（2014 年春季多省联考）经营性资产是在生产和流通中能够为社会提供商品或劳务的资产。经营性资产的使用单位是具有法人地位的企业，其运营要以追求经济效益为原则。从会计角度看，主要指企业因盈利目的而持有，且实际也具有盈利能力的资产。根据上述定义，下列不属于“经营性资产”的是：           
A．某电视机厂中积压的黑白电视机             
B．建设中尚未投入使用的水力发电站               
C．某外贸公司用于交纳税金的银行存款             
D．某重点中学新购入的实验室设备和器材


**解析**      
<BlurredAnswer>       
我们需要先根据题干定义来明确“经营性资产”的概念：这类资产在生产和流通中能够为社会提供商品或劳务，其使用单位是法人地位的企业，并且运营要追求经济效益。从会计角度看，主要是指企业因盈利目的而持有，且实际具有盈利能力的资产。

逐一分析每个选项：

A. 某电视机厂中积压的黑白电视机
- 虽然这些电视机是积压的，但它们仍然是可以出售的，可以带来经济效益。因此，它们仍然属于经营性资产。

B. 建设中尚未投入使用的水力发电站
- 尽管这个发电站尚未投入使用，但从长远来看，它将为社会提供电力，且具有经济效益。因此，它也属于经营性资产。

C. 某外贸公司用于交纳税金的银行存款
- 银行存款是企业持有的资金，用于企业运营和纳税，从而支持企业的盈利活动。因此，这些存款属于经营性资产。

D. 某重点中学新购入的实验室设备和器材
- 重点中学是一个公共服务机构，不以盈利为目的，而是为了教育。因此，尽管这些设备和器材用于教学，但它们不符合“经营性资产”的定义。

综上所述，**D. 某重点中学新购入的实验室设备和器材** 不属于“经营性资产”。
</BlurredAnswer>

## 2. 条件结果法
- 方式：通过，利用；      
- 原因：因为，由于；
- 时间：当……时；
- 结果：导致，从而；
- 目的：以，达到

（2017 北京）互动式教学，就是通过营造多边互动的教学环境，在教学双方平等交流探讨的过程中，达到不同观点碰撞交融，进而激发教学双方的主动性和探索性，从而提高教学效果的一种教学方式。根据上述定义，以下属于互动式教学的是（ ）。            
A.某学院通过文化讲座向外国留学生介绍儒家思想                
B.王老师带领教育专业的学生进行课堂教学模拟实验              
C.小王觉得自己英语不好，报名参加课外一对一辅导              
D.学生和老师就某一个问题展开课堂辩论            

**解析**     
<BlurredAnswer>       
根据题干定义，互动式教学具备以下特点：
- 营造多边互动的教学环境
- 教学双方平等交流探讨
- 达到不同观点的碰撞交融
- 激发教学双方的主动性和探索性
- 提高教学效果
逐一分析选项：

A. 某学院通过文化讲座向外国留学生介绍儒家思想
- 这是一个单向的传授知识形式，主要由讲师讲授，留学生听讲，缺乏多边互动和观点碰撞，不符合定义。

B. 王老师带领教育专业的学生进行课堂教学模拟实验
- 这是一种教师引导学生进行实践性活动的形式，虽然有互动，但更偏向于教师指导下的实践操作，未必涉及平等交流和观点碰撞，不符合定义。

C. 小王觉得自己英语不好，报名参加课外一对一辅导
- 一对一辅导通常是指导关系，不太涉及多边互动和平等交流，也不一定会有观点碰撞，不符合定义。

D. 学生和老师就某一个问题展开课堂辩论
- 课堂辩论是一种多边互动的教学活动，学生和老师可以平等交流探讨，不同观点碰撞，激发双方的主动性和探索性，符合定义。

因此，符合互动式教学定义的选项是D。
</BlurredAnswer>

A.某高校不断加强对外交流，派遣访问学者的名额逐年增加

A. 某高校不断加强对外交流，派遣访问学者的名额逐年增加。
- 这个选项的重点在于加强对外交流和增加访问学者名额。这主要是组织的外部合作和人员流动性问题，并没有明显体现通过规章制度和奖惩手段来约束成员的行为，因此不符合组织控制的定义。

- 这个选项主要是为了帮助新员工更快适应新环境，属于企业文化和培训活动的一部分，而不是通过规章制度和奖惩手段来约束成员的行为，因此也不符合组织控制的定义。

在C和D选项中，D选项更全面地体现了通过规章制度和奖惩手段来约束员工的行为，因此最能体现组织控制的定义。

综上所述，答案是D。

**例题**

我们先明确内隐学习的定义：内隐学习是无需意志努力的潜意识学习，特点是人在潜意识中获得知识，并能将其应用于特定任务中。因此，内隐学习不涉及刻意的学习和努力。

A. 小张在高考前做了大量数学习题，因此在数学考试中获得高分。

- 小李通过观看别人下象棋，在不知不觉中掌握了下象棋的方法，这是一个内隐学习的过程。

C. 儿童无需系统地学习语词和语法规则，不知不觉地就会说母语。
- 儿童在自然成长过程中，潜移默化地学会了母语，这是一个典型的内隐学习的例子。

- 这个孩子通过日常的耳濡目染，无需刻意学习就能唱京剧，这也符合内隐学习的特征。

综合分析，选项A中明确提到小张通过大量的刻意练习取得了高分，这与内隐学习的定义不符。因此，正确答案是A。
</BlurredAnswer>

**例题**       
（2017 国考）传递关系指的是对任意的元素 A、B、C 来说，若元素A 与元素 B 有某关系并且元素 B 与元素 C 有该关系，则元素 A 与元素 C 也有该关系。反传递关系指的是若元素 A 与元素 B 有某种关系并且元素 B 与元素 C 有该关系，但元素 A 与元素 C 没有该关系。根据上述定义，下列关系属于传递关系的是（）。     
A.自然数中的大于关系      
B.生活中的同学关系      
C.家庭中的父子关系      
D.食物链的天敌关系     

**解析**       
根据题目中给出的定义，传递关系和反传递关系的区别在于：

- **传递关系**：如果元素 A 与元素 B 有某种关系，且元素 B 与元素 C 有该关系，则元素 A 与元素 C 也有该关系。
- **反传递关系**：如果元素 A 与元素 B 有某种关系，且元素 B 与元素 C 有该关系，但元素 A 与元素 C 没有该关系。

逐项分析选项：

**A. 自然数中的大于关系**

在自然数中，如果 $ A > B $ 且 $ B > C $，那么根据传递性，可以推出 $ A > C $。因此，自然数中的“大于关系”符合传递关系的定义。

**B. 生活中的同学关系**

如果 A 是 B 的同学，B 是 C 的同学，这并不意味着 A 和 C 一定是同学。例如，A 和 B 可能是同一所学校的同学，而 B 和 C 是另一所学校的同学。因此，生活中的同学关系不符合传递关系。

**C. 家庭中的父子关系**

如果 A 是 B 的父亲，B 是 C 的父亲，那么 A 是 C 的祖父，而不是父亲。因此，家庭中的父子关系不符合传递关系，反而更像是反传递关系。

**D. 食物链的天敌关系**

如果 A 是 B 的天敌，B 是 C 的天敌，这并不意味着 A 是 C 的天敌。例如，A 可能是狼，B 是兔子，C 是草，狼吃兔子，兔子吃草，但狼不吃草。因此，食物链的天敌关系不符合传递关系。

综上所述，符合传递关系的选项是：

**A. 自然数中的大于关系**

因此，正确答案是 **A**。

## 5. 同构选项排除法 
同构选项——意思相似、结构相似的选项。
