# 平均数增长率

import BlurredAnswer from '@/components/ui/BlurredAnswer';

## 核心概念

### 什么是平均数增长率

平均数增长率是指某个平均指标（如人均GDP、亩产量、单位面积产值等）在两个时期之间的增长率。它反映了<span style={{color:'red'}}>效率型指标的变化趋势</span>。

### 生活化理解

想象一下班级考试成绩的例子：
- 基期：班级总分2400分，40人，平均分60分
- 现期：班级总分2700分，45人，平均分？增长率？

这就是典型的平均数增长率问题。我们需要分析的是<span style={{color:'red'}}>平均成绩是否提高了，提高了多少</span>。

### 核心公式

平均数增长率的核心公式为：

$$r = \frac{a - b}{1 + b}$$

其中：
- $r$ 为平均数的增长率
- $a$ 为分子的增长率（如总量增长率）
- $b$ 为分母的增长率（如人口或数量增长率）

### 公式推导

让我们用班级成绩的例子来推导这个公式：

**第一步：建立基本关系**

设基期平均数为：$X_0 = \frac{A_0}{B_0}$

设现期平均数为：$X_1 = \frac{A_1}{B_1}$

**第二步：定义增长率**

平均数的增长率定义为：
$$r = \frac{X_1 - X_0}{X_0}$$

**第三步：变形处理**

将平均数表达式代入：
$$r = \frac{\frac{A_1}{B_1} - \frac{A_0}{B_0}}{\frac{A_0}{B_0}}$$

化简得：
$$r = \frac{A_1}{B_1} \times \frac{B_0}{A_0} - 1$$

**第四步：引入增长率概念**

设$a = \frac{A_1 - A_0}{A_0}$（分子增长率），$b = \frac{B_1 - B_0}{B_0}$（分母增长率）

则：$A_1 = A_0(1 + a)$，$B_1 = B_0(1 + b)$

**第五步：最终推导**

代入得：
$$r = \frac{A_0(1 + a)}{B_0(1 + b)} \times \frac{B_0}{A_0} - 1$$

$$r = \frac{1 + a}{1 + b} - 1$$

$$r = \frac{1 + a - 1 - b}{1 + b} = \frac{a - b}{1 + b}$$

### 公式记忆技巧

1. <span style={{color:'red'}}>分子分母谁增长得快，平均数就向谁靠拢</span>
2. 当$a > b$时，平均数上升；当$a < b$时，平均数下降
3. <span style={{color:'orange'}}>注意分母有$(1 + b)$，不能简单用$a - b$</span>

## 真题讲解

### 主题一：人均类增长率问题

2023年，某市GDP总量为1800亿元，比上年增长12%；常住人口为600万人，比上年增长3%。问2023年该市人均GDP比上年约增长：

A. 8.7% <br />
B. 9.0% <br />
C. 9.3% <br />
D. 15.0% <br />

<BlurredAnswer>
**解题思路：**

**第一步：识别题型**
- 题目求的是"人均GDP增长率"，这是典型的平均数增长率问题
- 人均GDP = $\frac{GDP总量}{人口数}$

**第二步：确定参数**
- $a$ = GDP总量增长率 = 12% = 0.12
- $b$ = 人口增长率 = 3% = 0.03

**第三步：套用公式**
$$r = \frac{a - b}{1 + b} = \frac{0.12 - 0.03}{1 + 0.03} = \frac{0.09}{1.03}$$

**第四步：计算结果**
$$r = \frac{0.09}{1.03} \approx \frac{9}{103} \approx 8.7\%$$

**答案：A**

**解题关键：**
<span style={{color:'red'}}>人均类指标 = 总量 ÷ 人数，分子是总量，分母是人数</span>
</BlurredAnswer>

### 主题二：单位产量增长率问题

某农场2022年粮食总产量为2400吨，比上年增长8%；种植面积为300公顷，比上年增长5%。则2022年该农场每公顷产量比上年约增长：

A. 2.9% <br />
B. 3.0% <br />
C. 3.1% <br />
D. 13.0% <br />

<BlurredAnswer>
**解题思路：**

**第一步：识别题型**
- 求"每公顷产量增长率"，属于平均数增长率
- 每公顷产量 = $\frac{总产量}{种植面积}$

**第二步：确定参数**
- $a$ = 总产量增长率 = 8% = 0.08
- $b$ = 种植面积增长率 = 5% = 0.05

**第三步：套用公式**
$$r = \frac{a - b}{1 + b} = \frac{0.08 - 0.05}{1 + 0.05} = \frac{0.03}{1.05}$$

**第四步：计算结果**
$$r = \frac{0.03}{1.05} \approx \frac{3}{105} \approx 2.9\%$$

**答案：A**

**解题关键：**
<span style={{color:'red'}}>亩产、公顷产量等都是单位面积产量，分子是总产量，分母是面积</span>
</BlurredAnswer>

### 主题三：经济效益增长率问题

某企业2022年营业收入为5000万元，同比增长15%；员工人数为800人，同比增长6%。则2022年该企业人均营业收入比上年约增长：

A. 8.5% <br />
B. 8.8% <br />
C. 9.0% <br />
D. 21.0% <br />

<BlurredAnswer>
**解题思路：**

**第一步：识别题型**
- 求"人均营业收入增长率"，典型的平均数增长率问题
- 人均营业收入 = $\frac{营业收入}{员工人数}$

**第二步：确定参数**
- $a$ = 营业收入增长率 = 15% = 0.15
- $b$ = 员工人数增长率 = 6% = 0.06

**第三步：套用公式**
$$r = \frac{a - b}{1 + b} = \frac{0.15 - 0.06}{1 + 0.06} = \frac{0.09}{1.06}$$

**第四步：计算结果**
$$r = \frac{0.09}{1.06} \approx \frac{9}{106} \approx 8.5\%$$

**答案：A**

**解题关键：**
<span style={{color:'red'}}>人均收入、人均产值等经济指标，分子是经济总量，分母是人数</span>
</BlurredAnswer>

### 主题四：复合增长率问题

某地区2021年进出口贸易总额为180亿美元，比上年增长20%；贸易企业数量为450家，比上年增长12.5%。问2021年该地区平均每家企业进出口贸易额比上年约增长：

A. 6.7% <br />
B. 7.0% <br />
C. 7.5% <br />
D. 32.5% <br />

<BlurredAnswer>
**解题思路：**

**第一步：识别题型**
- 求"平均每家企业贸易额增长率"
- 平均每家企业贸易额 = $\frac{贸易总额}{企业数量}$

**第二步：确定参数**
- $a$ = 贸易总额增长率 = 20% = 0.20
- $b$ = 企业数量增长率 = 12.5% = 0.125

**第三步：套用公式**
$$r = \frac{a - b}{1 + b} = \frac{0.20 - 0.125}{1 + 0.125} = \frac{0.075}{1.125}$$

**第四步：计算结果**
$$r = \frac{0.075}{1.125} = \frac{75}{1125} = \frac{1}{15} \approx 6.7\%$$

**答案：A**

**解题关键：**
<span style={{color:'orange'}}>注意12.5%的小数表示是0.125，计算时要仔细</span>
</BlurredAnswer>

### 主题五：混合增长率问题

某市2023年社会消费品零售总额为2400亿元，比上年增长9%；常住人口为1200万人，比上年下降2%。则2023年该市人均社会消费品零售额比上年约增长：

A. 11.2% <br />
B. 11.5% <br />
C. 11.8% <br />
D. 7.0% <br />

<BlurredAnswer>
**解题思路：**

**第一步：识别题型**
- 求"人均消费品零售额增长率"
- 人均消费品零售额 = $\frac{零售总额}{人口数}$

**第二步：确定参数**
- $a$ = 零售总额增长率 = 9% = 0.09
- $b$ = 人口增长率 = -2% = -0.02（注意是负增长）

**第三步：套用公式**
$$r = \frac{a - b}{1 + b} = \frac{0.09 - (-0.02)}{1 + (-0.02)} = \frac{0.09 + 0.02}{0.98} = \frac{0.11}{0.98}$$

**第四步：计算结果**
$$r = \frac{0.11}{0.98} \approx \frac{11}{98} \approx 11.2\%$$

**答案：A**

**解题关键：**
<span style={{color:'red'}}>当分母是负增长时，$a - b$变成$a - (-b) = a + b$，分母$(1 + b)$中$b$仍为负数</span>
</BlurredAnswer>

## 技巧总结

### 解题步骤模板

1. **第一步：识别题型**
   - 关键词：人均、每...、单位...、平均每...
   - 时间：两个时期比较
   - 问法：增长率、增长了...%

2. **第二步：确定平均数结构**
   - 明确分子是什么（总量）
   - 明确分母是什么（个数、面积、人数等）

3. **第三步：提取增长率**
   - $a$：分子的增长率
   - $b$：分母的增长率

4. **第四步：套用公式计算**
   - $r = \frac{a - b}{1 + b}$

### 快速计算技巧

1. **当$b$很小时的近似**
   - 当$|b| < 5\%$时，可以近似：$r \approx a - b$
   - <span style={{color:'orange'}}>但考试中建议精确计算，避免误差</span>

2. **分数计算技巧**
   - 将百分数转为分数：12.5% = 1/8，6.25% = 1/16
   - 利用约分简化计算

3. **估算方法**
   - 先算$a - b$的大致值
   - 再考虑$(1 + b)$的影响，通常使结果略小于$a - b$

### 常见易错点

1. <span style={{color:'orange'}}>混淆分子分母</span>
   - 人均GDP：分子是GDP，分母是人口
   - 亩产量：分子是总产量，分母是面积

2. <span style={{color:'orange'}}>忘记公式中的$(1 + b)$</span>
   - 不能简单用$a - b$
   - 必须除以$(1 + b)$

3. <span style={{color:'orange'}}>负增长的处理</span>
   - 负增长时$b$为负数
   - $a - b$变成$a - (-|b|) = a + |b|$
   - $(1 + b)$变成$(1 - |b|)$

4. <span style={{color:'orange'}}>单位换算错误</span>
   - 注意万人、千人等单位
   - 面积单位：公顷、亩、平方公里

### 秒杀判断技巧

1. **升降判断**
   - $a > b$：平均数上升
   - $a < b$：平均数下降
   - $a = b$：平均数不变

2. **数量级判断**
   - 平均数增长率通常在$|a - b|$附近
   - 一般小于较大的那个增长率

3. **选项排除**
   - 排除明显过大或过小的选项
   - 平均数增长率不会超过分子增长率
