# 基期平均数

## 核心概念

### 什么是基期平均数？

基期平均数是资料分析中的重要概念，指的是<span style={{color:'red'}}>基期总量与基期总个数的比值</span>。

让我们用一个生活中的例子来理解：

假设小明开了一家奶茶店，现在是2024年（现期），去年是2023年（基期）。我们想知道2023年平均每杯奶茶的价格。

- 如果2023年总销售额是10万元（基期总量）
- 2023年总共卖出了2万杯奶茶（基期总个数）
- 那么2023年平均每杯奶茶的价格 = 10万元 ÷ 2万杯 = 5元/杯

### 核心公式推导

**基础公式：**
$$\text{基期平均数} = \frac{\text{基期总量}}{\text{基期总个数}}$$

但是，在实际题目中，我们通常只知道现期数据和增长率，所以需要推导出实用公式。

**推导过程：**

设现期总量为 $A$，增长率为 $a\%$，现期总个数为 $B$，增长率为 $b\%$

根据基期与现期的关系：
- 基期总量 = $\frac{A}{1+a\%}$
- 基期总个数 = $\frac{B}{1+b\%}$

因此：
$$\text{基期平均数} = \frac{\frac{A}{1+a\%}}{\frac{B}{1+b\%}} = \frac{A}{B} \times \frac{1+b\%}{1+a\%}$$

**最终公式：**
$$\text{基期平均数} = \text{现期平均数} \times \frac{1+\text{总个数增长率}}{1+\text{总量增长率}}$$

其中，现期平均数 = $\frac{A}{B}$

### 间隔基期型（隔年问题）

当题目涉及隔年数据时，需要先求出间隔增长率：

**间隔增长率公式：**
$$R_{\text{间隔}} = r_1 + r_2 + r_1 \times r_2$$

其中 $r_1$ 是第一年增长率，$r_2$ 是第二年增长率。

**间隔基期平均数：**
$$\text{间隔基期平均数} = \frac{\text{现期平均数}}{1 + R_{\text{间隔}}}$$

## 真题讲解

### 主题一：基础公式型问题

**例1**（2022年国考）：
2021年某地区小麦产量1200万吨，同比增长8%；种植面积400万公顷，同比增长5%。问2020年平均每公顷产量是多少？

A. 2.6吨/公顷 <br />
B. 2.8吨/公顷 <br />
C. 3.0吨/公顷 <br />
D. 3.2吨/公顷 <br />

<BlurredAnswer>
**解题步骤：**

**第一步：识别题型**
- 求基期平均数（2020年是基期，2021年是现期）
- 总量：小麦产量，总个数：种植面积

**第二步：列出已知条件**
- 现期总量：1200万吨，增长率：8%
- 现期总个数：400万公顷，增长率：5%

**第三步：套用公式**
$$\text{基期平均数} = \frac{\text{现期总量}}{\text{现期总个数}} \times \frac{1+\text{总个数增长率}}{1+\text{总量增长率}}$$

$$= \frac{1200}{400} \times \frac{1+5\%}{1+8\%} = 3 \times \frac{1.05}{1.08}$$

**第四步：计算结果**
$$= 3 \times \frac{1.05}{1.08} = 3 \times 0.972 = 2.916$$

**第五步：选择答案**
约等于2.9吨/公顷，最接近选项B：2.8吨/公顷

**答案：B**

2020年某市GDP总量为8000亿元，同比增长6%；人口为500万人，同比增长2%。问2019年人均GDP是多少？

A. 14.2万元/人 <br />
B. 15.4万元/人 <br />
C. 16.8万元/人 <br />
D. 17.5万元/人 <br />

<BlurredAnswer>
**解题步骤：**

**第一步：识别题型**
- 求基期平均数（2019年是基期，2020年是现期）
- 总量：GDP总量，总个数：人口

**第二步：列出已知条件**
- 现期总量：8000亿元，增长率：6%
- 现期总个数：500万人，增长率：2%

**第三步：套用公式**
$$\text{基期平均数} = \frac{8000}{500} \times \frac{1+2\%}{1+6\%} = 16 \times \frac{1.02}{1.06}$$

**第四步：计算结果**
$$= 16 \times \frac{1.02}{1.06} = 16 \times 0.962 = 15.39$$

**第五步：选择答案**
约等于15.4万元/人

### 主题二：间隔基期型问题

**例3**（2023年国考）：
2022年某省工业增加值3600亿元，比2021年增长5%，2021年比2020年增长3%。问2020年该省工业增加值是多少？

A. 3300亿元 <br />
B. 3400亿元 <br />
C. 3500亿元 <br />
D. 3600亿元 <br />

<BlurredAnswer>
**解题步骤：**

**第一步：识别题型**
- 间隔基期问题（2020年是间隔基期，2022年是现期）
- 需要先求间隔增长率

**第二步：计算间隔增长率**
$$R_{\text{间隔}} = r_1 + r_2 + r_1 \times r_2$$
$$= 3\% + 5\% + 3\% \times 5\%$$
$$= 8\% + 0.15\% = 8.15\%$$

**第三步：套用间隔基期公式**
$$\text{间隔基期值} = \frac{\text{现期值}}{1 + R_{\text{间隔}}}$$
$$= \frac{3600}{1 + 8.15\%} = \frac{3600}{1.0815} = 3328$$

**第四步：选择答案**
约等于3300亿元

**答案：A**
</BlurredAnswer>

### 主题三：增长率相等的特殊情况

**例4**（2024年省考）：
2023年某市居民人均可支配收入48000元，同比增长7%；人口总数160万人，同比增长7%。问2022年该市居民人均可支配收入是多少？

A. 44860元 <br />
B. 45000元 <br />
C. 46000元 <br />
D. 48000元 <br />

<BlurredAnswer>
**解题步骤：**

**第一步：识别题型**
- 基期平均数问题
- <span style={{color:'orange'}}>注意：总量和总个数的增长率相等</span>

**第二步：特殊情况判断**
当总量增长率 = 总个数增长率时：
$$\text{基期平均数} = \text{现期平均数} \times \frac{1+b\%}{1+a\%} = \text{现期平均数} \times \frac{1+7\%}{1+7\%} = \text{现期平均数}$$

**第三步：得出结论**
由于增长率相等，基期平均数 = 现期平均数 = 48000元

**第四步：验证**
这是一个重要的<span style={{color:'red'}}>秒杀技巧</span>：当总量和总个数增长率相等时，基期平均数等于现期平均数。

**答案：D**
</BlurredAnswer>

### 主题四：比较类问题

**例5**（2023年联考）：
2022年A市人均GDP为8.5万元，GDP增长率为6%，人口增长率为2%；B市人均GDP为8.2万元，GDP增长率为8%，人口增长率为3%。问2021年哪个市的人均GDP更高？

A. A市 <br />
B. B市 <br />
C. 两市相等 <br />
D. 无法确定 <br />

<BlurredAnswer>
**解题步骤：**

**第一步：识别题型**
- 基期平均数比较问题
- 需要分别计算两市2021年人均GDP

**第二步：计算A市2021年人均GDP**
$$\text{A市基期人均GDP} = 8.5 \times \frac{1+2\%}{1+6\%} = 8.5 \times \frac{1.02}{1.06} = 8.5 \times 0.962 = 8.18$$

**第三步：计算B市2021年人均GDP**
$$\text{B市基期人均GDP} = 8.2 \times \frac{1+3\%}{1+8\%} = 8.2 \times \frac{1.03}{1.08} = 8.2 \times 0.954 = 7.82$$

**第四步：比较结果**
A市：8.18万元 > B市：7.82万元

**第五步：快速判断技巧**
当现期平均数相近时，可以直接比较 $\frac{1+b\%}{1+a\%}$ 的大小：
- A市：$\frac{1.02}{1.06} = 0.962$
- B市：$\frac{1.03}{1.08} = 0.954$

**答案：A**
</BlurredAnswer>

## 技巧总结

### 技巧一：选项差距法

当选项差距较大（>10%）时，可以使用估算：

**快速估算公式：**
$$\text{基期平均数} \approx \frac{A}{B} \times [1 + (b\% - a\%)]$$

其中 $a\%$ 是总量增长率，$b\%$ 是总个数增长率。

**例6**：
现期平均数为100，总量增长率为20%，总个数增长率为10%。

**精确计算：**
$$\text{基期平均数} = 100 \times \frac{1+10\%}{1+20\%} = 100 \times \frac{1.1}{1.2} = 91.67$$

**快速估算：**
$$\text{基期平均数} \approx 100 \times [1 + (10\% - 20\%)] = 100 \times 0.9 = 90$$

<span style={{color:'orange'}}>注意：这种方法只适用于选项差距较大的情况</span>

### 技巧二：增长率相等秒杀

当总量增长率 = 总个数增长率时：
$$\text{基期平均数} = \text{现期平均数}$$

这是因为：
$$\frac{1+b\%}{1+a\%} = \frac{1+a\%}{1+a\%} = 1$$

### 技巧三：比较类题目速判

比较两个基期平均数的大小时：
1. **第一步：**比较现期平均数 $\frac{A_1}{B_1}$ 与 $\frac{A_2}{B_2}$
2. **第二步：**比较调整系数 $\frac{1+b_1\%}{1+a_1\%}$ 与 $\frac{1+b_2\%}{1+a_2\%}$
3. **第三步：**综合判断

**判断规律：**
- 如果现期平均数差距很大，且调整系数接近，则现期大的基期也大
- 如果现期平均数接近，则主要看调整系数的大小

### 技巧四：间隔基期速算

对于间隔基期问题，当增长率较小时：
$$R_{\text{间隔}} \approx r_1 + r_2$$

<span style={{color:'orange'}}>注意：这种近似只适用于增长率都小于10%的情况</span>

**例7**：
$r_1 = 3\%$，$r_2 = 5\%$

**精确计算：**
$$R_{\text{间隔}} = 3\% + 5\% + 3\% \times 5\% = 8.15\%$$

**快速估算：**
$$R_{\text{间隔}} \approx 3\% + 5\% = 8\%$$

### 技巧五：分数化简技巧

在计算 $\frac{1+b\%}{1+a\%}$ 时，可以先化简分数：

**例8**：
$$\frac{1+8\%}{1+12\%} = \frac{1.08}{1.12} = \frac{108}{112} = \frac{27}{28}$$

然后利用分数的性质：$\frac{27}{28} = 1 - \frac{1}{28} \approx 1 - 3.6\% = 0.964$

这样可以避免直接除法运算，提高计算速度。

### 常见错误提醒

1. **<span style={{color:'red'}}>易错点</span>**：混淆基期和现期的概念
2. **<span style={{color:'red'}}>易错点</span>**：颠倒分子分母的位置
3. **<span style={{color:'red'}}>易错点</span>**：忘记区分总量和总个数的增长率
4. **<span style={{color:'red'}}>易错点</span>**：间隔基期问题中忘记计算间隔增长率

### 解题流程总结

1. **识别题型**：确定是基期平均数还是间隔基期平均数
2. **明确量值**：区分总量、总个数及其增长率
3. **套用公式**：根据题型选择合适的公式
4. **计算结果**：注意计算精度和选项差距
5. **验证答案**：检查结果的合理性
