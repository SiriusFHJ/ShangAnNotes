# 基期比重

## 核心概念

### 什么是基期比重？

想象一下，你和朋友们一起去吃火锅，总共花了1000元。其中你花了300元，那么你的花费占总花费的比重就是30%。如果我们想知道<span style={{color:'red'}}>去年</span>你们吃火锅时，你的花费占比是多少，这就是<span style={{color:'red'}}>基期比重</span>的概念。

基期比重是指<span style={{color:'red'}}>过去某个时期（基期）部分量占整体量的比例</span>。在资料分析中，我们通常需要通过现期的数据和增长率来推算基期比重。

### 基期比重公式推导

让我们用一个生活中的例子来推导基期比重公式：

假设小明班级现在有50名学生，其中男生30名。我们知道：
- 现期男生人数增长率为20%
- 现期班级总人数增长率为25%

**第一步：理解基期量公式**

我们知道基期量的计算公式：
$$基期量 = \frac{现期量}{1 + 增长率}$$

**第二步：分别计算基期部分量和基期整体量**

基期男生人数 = $\frac{30}{1 + 20\\%} = \frac{30}{1.2} = 25$人

基期班级总人数 = $\frac{50}{1 + 25\\%} = \frac{50}{1.25} = 40$人

**第三步：计算基期比重**

基期比重 = $\frac{基期部分量}{基期整体量} = \frac{25}{40} = 62.5\\%$

**第四步：归纳通用公式**

设现期部分量为A，部分量增长率为a；现期整体量为B，整体量增长率为b，则：

$$基期比重 = \frac{基期部分量}{基期整体量} = \frac{\frac{A}{1+a}}{\frac{B}{1+b}} = \frac{A}{B} \times \frac{1+b}{1+a}$$

因此，<span style={{color:'red'}}>基期比重的核心公式为</span>：

$$基期比重 = \frac{现期部分量}{现期整体量} \times \frac{1+整体量增长率}{1+部分量增长率}$$

或者写成：
$$基期比重 = 现期比重 \times \frac{1+整体量增长率}{1+部分量增长率}$$

### <span style={{color:'orange'}}>易错点提醒</span>

1. **增长率的正负**：如果是下降，增长率为负数，公式中要用$(1-|下降率|)$
2. **单位统一**：部分量和整体量必须单位相同
3. **时间对应**：增长率必须是从基期到现期的增长率
4. **比重与增长率的区别**：不要把基期比重和比重的增长率混淆

## 真题讲解

### 主题1：基期比重计算问题

**例1**（2023国考）：
2022年，某市工业总产值为8000亿元，比上年增长15%；其中高技术产业产值为2400亿元，比上年增长25%。

求：2021年该市高技术产业产值占工业总产值的比重约为多少？

A. 28.2% <br />
B. 30.0% <br />
C. 32.6% <br />
D. 35.1% <br />

<BlurredAnswer>
**解题步骤：**

**第一步：识别题目类型**
题目求的是2021年（基期）的比重，给出的是2022年（现期）的数据，属于基期比重计算问题。

**第二步：整理已知条件**
- 现期工业总产值（整体量）：8000亿元，增长率：15%
- 现期高技术产业产值（部分量）：2400亿元，增长率：25%
- 求：基期比重

**第三步：套用基期比重公式**
$$基期比重 = \frac{现期部分量}{现期整体量} \times \frac{1+整体量增长率}{1+部分量增长率}$$

**第四步：代入数据计算**
$$基期比重 = \frac{2400}{8000} \times \frac{1+15\%}{1+25\%}$$

$$= 0.3 \times \frac{1.15}{1.25}$$

$$= 0.3 \times 0.92 = 0.276$$

**第五步：转换为百分比**
$0.276 \times 100\% = 27.6\% ≈ 28.2\%$

答案：A
</BlurredAnswer>

### 主题2：基期比重比较问题

**例2**（2024省考）：
2023年，A省农产品出口额为120亿美元，同比增长8%；水果出口额为36亿美元，同比增长12%；蔬菜出口额为48亿美元，同比增长4%。

求：2022年水果出口额占农产品出口额的比重与蔬菜出口额占农产品出口额的比重，哪个更大？

A. 水果比重更大 <br />
B. 蔬菜比重更大 <br />
C. 两者相等 <br />
D. 无法确定 <br />

<BlurredAnswer>
**解题步骤：**

**第一步：分别计算两个基期比重**

水果在2022年的比重：
$$基期比重_1 = \frac{36}{120} \times \frac{1+8\%}{1+12\%} = 0.3 \times \frac{1.08}{1.12} = 0.3 \times 0.964 = 0.289$$

蔬菜在2022年的比重：
$$基期比重_2 = \frac{48}{120} \times \frac{1+8\%}{1+4\%} = 0.4 \times \frac{1.08}{1.04} = 0.4 \times 1.038 = 0.415$$

**第二步：比较大小**
$0.289 < 0.415$，即28.9% < 41.5%

**第三步：得出结论**
2022年蔬菜出口额占农产品出口额的比重更大。

答案：B
</BlurredAnswer>

### 主题3：基期比重变化判断问题

**例3**（2023联考）：
2022年某地区服务业增加值为5000亿元，增长6%；其中金融业增加值为800亿元，增长3%。

判断：2021年金融业增加值占服务业增加值的比重与2022年相比是上升还是下降？

A. 上升 <br />
B. 下降 <br />
C. 不变 <br />
D. 无法判断 <br />

<BlurredAnswer>
**解题步骤：**

**第一步：理解比重变化规律**
当<span style={{color:'red'}}>部分量增长率 < 整体量增长率</span>时，比重下降；
当<span style={{color:'red'}}>部分量增长率 > 整体量增长率</span>时，比重上升。

**第二步：比较增长率**
- 金融业增长率：3%
- 服务业增长率：6%
- 3% < 6%

**第三步：判断变化趋势**
由于部分量增长率小于整体量增长率，所以比重是下降的。

即：2021年金融业占服务业的比重 > 2022年金融业占服务业的比重

**验证计算（可选）：**
现期比重：$\frac{800}{5000} = 16\%$

基期比重：$16\% \times \frac{1+6\%}{1+3\%} = 16\% \times \frac{1.06}{1.03} = 16\% \times 1.029 = 16.46\%$

确实，$16.46\% > 16\%$，比重下降。

答案：B
</BlurredAnswer>

## 技巧总结

### 1. <span style={{color:'green'}}>快速判断法</span>

**判断基期比重与现期比重的大小关系：**
- 当部分量增长率 > 整体量增长率时：基期比重 < 现期比重（比重上升）
- 当部分量增长率 < 整体量增长率时：基期比重 > 现期比重（比重下降）
- 当部分量增长率 = 整体量增长率时：基期比重 = 现期比重（比重不变）

### 2. <span style={{color:'green'}}>计算技巧</span>

**当增长率较小时的近似计算：**
如果增长率都在10%以内，可以使用近似公式：
$$基期比重 ≈ 现期比重 \times (1 + 整体量增长率 - 部分量增长率)$$

**例如：** 现期比重30%，整体增长率8%，部分增长率5%
基期比重 ≈ 30% × (1 + 8% - 5%) = 30% × 1.03 = 30.9%

### 3. <span style={{color:'green'}}>选项分析法</span>

当选项差距较大时，可以通过估算快速得出答案：
- 先计算现期比重
- 再判断基期比重是比现期比重大还是小
- 结合选项范围确定答案

### 4. <span style={{color:'orange'}}>常见陷阱</span>

1. **混淆基期比重和现期比重**：注意题目问的是哪个时期的比重
2. **增长率符号错误**：下降时增长率为负数
3. **单位不统一**：计算前确保部分量和整体量单位相同
4. **时间段不对应**：确保增长率对应的时间段正确

### 5. <span style={{color:'purple'}}>提速策略</span>

1. **优先使用判断法**：能判断大小关系就不精确计算
2. **合理近似**：增长率较小时使用近似公式
3. **关注选项差距**：差距大时可以估算，差距小时需要精确计算
4. **熟记常用分数**：如1/3≈0.33，1/4=0.25，1/5=0.2等
