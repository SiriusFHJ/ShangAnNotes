# 增长量

## 核心概念

### 增长量的定义

增长量是指<span style={{color:'red'}}>某一时期相对于基期的绝对增长数值</span>，它反映的是数量的变化幅度。

**识别标志**：增长 + 具体单位（人/元/吨/万个等）

让我们用一个生活中的例子来理解增长量：

**例子**：小明2023年12月的工资是100元，2024年12月的工资是120元。

那么，2024年12月比2023年12月增长了多少元？

很显然：120 - 100 = 20元

这个20元就是<span style={{color:'red'}}>增长量</span>。

### 增长量的基本公式推导

#### 公式一：直接计算法

$$\text{增长量} = \text{现期量} - \text{基期量}$$

这是最直观的计算方法，当我们知道现期量和基期量时直接相减即可。

#### 公式二：基期×增长率法

设基期量为 $A$，增长率为 $r$，则：

现期量 = 基期量 $\times$ (1 + 增长率) = $A \times (1 + r)$

因此：
$$\text{增长量} = \text{现期量} - \text{基期量} = A \times (1 + r) - A = A \times r$$

即：$$\text{增长量} = \text{基期量} \times \text{增长率}$$

#### 公式三：现期量转换法

由于现期量 = 基期量 × (1 + 增长率)，可得：

$$\text{基期量} = \frac{\text{现期量}}{1 + \text{增长率}}$$

代入公式二：
$$\text{增长量} = \frac{\text{现期量}}{1 + r} \times r = \frac{\text{现期量} \times r}{1 + r}$$

**三个公式的关系**：
- 公式一：已知现期、基期时使用
- 公式二：已知基期、增长率时使用  
- 公式三：已知现期、增长率时使用

## 真题讲解

### 主题一：现期量与基期量直接计算

**考查形式**：直接给出现期量和基期量，求增长量

**解题步骤**：
1. 识别现期量和基期量
2. 直接使用公式：增长量 = 现期量 - 基期量
3. 注意计算技巧：划线减法、尾数法

**例1**：（2022广东）2020年，全国基层群众性自治组织较上年减少（）万个。

根据材料：2019年村委会53.3万个，居委会11万个；2020年村委会50.2万个，居委会11.3万个。

A. 1.4 <br />
B. 2.8 <br />
C. 4.2 <br />
D. 5.6 <br />

<BlurredAnswer>
**解题步骤**：

第一步：计算2019年（基期）总量
$$2019年总量 = 53.3 + 11 = 64.3\text{万个}$$

第二步：计算2020年（现期）总量
$$2020年总量 = 50.2 + 11.3 = 61.5\text{万个}$$

第三步：计算增长量（此题为减少量）
$$\text{增长量} = 61.5 - 64.3 = -2.8\text{万个}$$

由于结果为负数，说明减少了2.8万个。

**答案：B**
</BlurredAnswer>

### 主题二：现期量与增长率计算

**考查形式**：给出现期量和增长率，求增长量

**解题步骤**：
1. 识别现期量和增长率
2. 应用公式：增长量 = (现期量 × 增长率) ÷ (1 + 增长率)
3. <span style={{color:'red'}}>重点掌握百化分技巧</span>

**例2**：（2024江苏事业单位）全年城镇新增就业138.29万人，比上年增长5.1%。2023年江苏城镇新增就业人数同比增加多少万人？

A. 6.5 <br />
B. 6.7 <br />
C. 7.2 <br />
D. 7.8 <br />

<BlurredAnswer>
**解题步骤**：

第一步：识别已知量
- 现期量：138.29万人
- 增长率：5.1%

第二步：应用增长量公式
$$\text{增长量} = \frac{\text{现期量} \times r}{1 + r} = \frac{138.29 \times 5.1\%}{1 + 5.1\%}$$

第三步：<span style={{color:'red'}}>使用百化分技巧</span>
由于5.1%接近5%，而$5\% = \frac{1}{20}$

当$r = \frac{1}{20}$时，$1 + r = 1 + \frac{1}{20} = \frac{21}{20}$

$$\text{增长量} = \frac{138.29 \times \frac{1}{20}}{\frac{21}{20}} = \frac{138.29}{21} \approx 6.7\text{万人}$$

**答案：B**
</BlurredAnswer>

### 主题三：百化分速算技巧

**核心思想**：将百分数转换为分数，简化计算

**常用百化分对应关系**：

| 百分数 | 分数 | 百分数 | 分数 |
|--------|------|--------|------|
| 50% | 1/2 | 12.5% | 1/8 |
| 33.3% | 1/3 | 11.1% | 1/9 |
| 25% | 1/4 | 16.7% | 1/6 |
| 20% | 1/5 | 14.3% | 1/7 |
| 10% | 1/10 | 8.3% | 1/12 |

**例3**：2024年某市GDP为1200亿元，同比增长25%。求2024年该市GDP增长量。

A. 200亿元 <br />
B. 240亿元 <br />
C. 280亿元 <br />
D. 300亿元 <br />

<BlurredAnswer>
**解题步骤**：

第一步：识别已知量
- 现期量：1200亿元
- 增长率：25%

第二步：<span style={{color:'red'}}>百化分转换</span>
$25\% = \frac{1}{4}$

第三步：应用速算公式
当$r = \frac{1}{n}$时，$\text{增长量} = \frac{\text{现期量}}{n+1}$

$$\text{增长量} = \frac{1200}{4+1} = \frac{1200}{5} = 240\text{亿元}$$

**答案：B**
</BlurredAnswer>

### 主题四：增长量比较

**考查形式**：比较不同主体的增长量大小

**解题原则**：
1. **大大则大**：现期量大且增长率大，则增长量大
2. **一大一小**：需要进行具体计算比较

**例4**：2024年某省三个城市经济数据如下：
- A市：GDP 800亿元，增长率20%
- B市：GDP 600亿元，增长率30%  
- C市：GDP 900亿元，增长率15%

问：哪个城市GDP增长量最大？

A. A市 <br />
B. B市 <br />
C. C市 <br />
D. 无法判断 <br />

<BlurredAnswer>
**解题步骤**：

第一步：分析各城市情况
- A市：现期量较大，增长率中等
- B市：现期量中等，增长率最大
- C市：现期量最大，增长率最小

第二步：<span style={{color:'red'}}>应用百化分快速计算</span>
- A市：$20\% = \frac{1}{5}$，增长量 = $\frac{800}{5+1} = \frac{800}{6} ≈ 133.3$亿元
- B市：$30\% = \frac{3}{10}$，增长量 = $\frac{600 \times 3}{10+3} = \frac{1800}{13} ≈ 138.5$亿元
- C市：$15\% = \frac{3}{20}$，增长量 = $\frac{900 \times 3}{20+3} = \frac{2700}{23} ≈ 117.4$亿元

第三步：比较结果
B市增长量最大

**答案：B**
</BlurredAnswer>

### 主题五：增长量的变形应用

#### 拉动增长率

**定义**：部分量的增长对整体增长的贡献程度

**公式**：$$\text{拉动增长率} = \frac{\text{部分量的增长量}}{\text{整体量的基期量}}$$

**例5**：2024年某省GDP为5000亿元，比上年增长200亿元。其中工业增加值1500亿元，同比增长7.1%。问：工业拉动GDP增长了多少个百分点？

A. 1.5 <br />
B. 2.0 <br />
C. 2.5 <br />
D. 3.0 <br />

<BlurredAnswer>
**解题步骤**：

第一步：计算工业增加值的基期量
$$\text{工业基期量} = \frac{1500}{1 + 7.1\%} = \frac{1500}{1.071} ≈ 1400\text{亿元}$$

第二步：计算工业增长量
$$\text{工业增长量} = 1500 - 1400 = 100\text{亿元}$$

第三步：计算整体GDP基期量
$$\text{GDP基期量} = 5000 - 200 = 4800\text{亿元}$$

第四步：计算拉动增长率
$$\text{拉动增长率} = \frac{100}{4800} = \frac{1}{48} ≈ 2.1\%$$

**答案：B**
</BlurredAnswer>

#### 年均增长量

**定义**：多年期间平均每年的增长量

**公式**：$$\text{年均增长量} = \frac{\text{现期量} - \text{基期量}}{n}$$

<span style={{color:'orange'}}>注意：年份差的确定要根据具体情况，国考省考通常为实际年份差，江苏省考可能基期前推一年</span>

**例6**：2020年某市人口100万人，2024年120万人。求2020-2024年该市人口年均增长多少万人？

A. 4万人 <br />
B. 5万人 <br />
C. 6万人 <br />
D. 7万人 <br />

<BlurredAnswer>
**解题步骤**：

第一步：确定基期量和现期量
- 基期量（2020年）：100万人
- 现期量（2024年）：120万人

第二步：确定年份差
$$n = 2024 - 2020 = 4\text{年}$$

第三步：计算年均增长量
$$\text{年均增长量} = \frac{120 - 100}{4} = \frac{20}{4} = 5\text{万人}$$

**答案：B**
</BlurredAnswer>

## 技巧总结

### 1. 百化分记忆技巧

**基础五个（必会）**：
- $50\% = \frac{1}{2}$
- $33.3\% = \frac{1}{3}$  
- $25\% = \frac{1}{4}$
- $20\% = \frac{1}{5}$
- $10\% = \frac{1}{10}$

**"7-12"记忆法**：
整数部分+分母=20
- $12.5\% = \frac{1}{8}$ (12+8=20)
- $11.1\% = \frac{1}{9}$ (11+9=20)
- $9.1\% = \frac{1}{11}$ (9+11=20)
- $8.3\% = \frac{1}{12}$ (8+12=20)
- $7.7\% = \frac{1}{13}$ (7+13=20)

### 2. 增长量计算速算技巧

**当增长率为正且较小时**（r&lt;5%）：
$$\text{增长量} ≈ \text{现期量} \times r$$

**当增长率为标准百分数时**：
$$\text{增长量} = \frac{\text{现期量}}{n+1}$$
其中n为增长率对应分数的分母

### 3. 增长量比较技巧

**快速判断法**：
1. **大大则大**：现期量大且增长率大，则增长量必然大
2. **现期量相近**：增长率大的增长量大
3. **增长率相近**：现期量大的增长量大
4. **一大一小**：需要具体计算

### 4. 常见易错点

<span style={{color:'orange'}}>1. 混淆增长量与增长率的概念</span>
- 增长量有具体单位（人、元、吨等）
- 增长率用百分数表示

<span style={{color:'orange'}}>2. 年份差计算错误</span>
- 一般情况：年份差=现期年份-基期年份
- 特殊情况：如江苏省考可能基期前推一年

<span style={{color:'orange'}}>3. 负增长的理解</span>
- 负增长量表示减少
- 计算时要注意符号

<span style={{color:'orange'}}>4. 百化分应用错误</span>
- 必须先确认增长率可以准确转换为分数
- 近似值要根据选项差距决定是否可用

### 5. 解题建议

1. **读题时**：重点识别"增长"+"具体单位"的关键词
2. **列式时**：根据已知条件选择合适的公式
3. **计算时**：优先考虑百化分等速算技巧
4. **检验时**：结果要符合常理，注意数量级
