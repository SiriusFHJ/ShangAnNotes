# 间隔增长率专题详解

间隔增长率是资料分析中的一个重要考点，它解决的是"现期与间隔基期之间的增长关系"问题。掌握间隔增长率，不仅能快速解决相关题目，更能为复合增长、多期平均增长率等高难度题型打下坚实基础。

## 一、核心知识点梳理

### 1.1 间隔基期定义

想象一下你追女朋友的过程：

> **2021年**：你还是单身狗（间隔基期） <br />
> **2022年**：你开始追求心仪的女孩（中间期） <br />
> **2023年**：你们确定了恋爱关系（现期）

在这个浪漫的例子中：
- <span style={{color:'red'}}>**现期**</span>：2023年（我们分析的当前时点）
- <span style={{color:'red'}}>**间隔基期**</span>：2021年（与现期相隔一年的基期）
- <span style={{color:'red'}}>**中间期**</span>：2022年（连接现期和间隔基期的桥梁）

**间隔基期的核心特征**：
- 与现期年份<span style={{color:'red'}}>相隔一年</span>的基期年份
- 中间必须有一个"桥梁"年份
- 是计算两年总体增长的起始点

**数学表达**：
$$间隔基期量 = \frac{现期量}{1 + 间隔增长率}$$

### 1.2 间隔增长率计算公式推导

我们继续用追女朋友的例子来推导核心公式：

**设定条件**：
- 2021年你的月收入：5000元（间隔基期）
- 2022年涨薪：增长20%，月收入6000元（中间期）
- 2023年再涨薪：增长25%，月收入7500元（现期）

**推导过程**：

**第一步：理解连续增长关系** <br />
$$2022年收入 = 2021年收入 \times (1 + r_1)$$ <br />
$$2023年收入 = 2022年收入 \times (1 + r_2)$$ <br />

其中：
- $r_1 = 20\%$（2022年相对2021年的增长率）
- $r_2 = 25\%$（2023年相对2022年的增长率）

**第二步：建立现期与间隔基期的直接关系** <br />
$$2023年收入 = 2021年收入 \times (1 + r_1) \times (1 + r_2)$$ <br />

代入具体数值验证：<br />
$$7500 = 5000 \times (1 + 20\%) \times (1 + 25\%)$$ <br />
$$7500 = 5000 \times 1.2 \times 1.25 = 5000 \times 1.5$$ <br />

**第三步：推导间隔增长率公式** <br />
设间隔增长率为 $R$，则： <br />
$$2023年收入 = 2021年收入 \times (1 + R)$$ <br />
$$7500 = 5000 \times (1 + R)$$ <br />
$$(1 + R) = \frac{7500}{5000} = 1.5$$ <br />
$$R = 0.5 = 50\%$$

**第四步：建立通用公式**
根据：$(1 + R) = (1 + r_1) \times (1 + r_2)$

展开右边：
$$(1 + R) = 1 + r_1 + r_2 + r_1 \times r_2$$

因此得到<span style={{color:'red'}}>间隔增长率核心公式</span>：
$$R = r_1 + r_2 + r_1 \times r_2$$

验证：$R = 20\% + 25\% + 20\% \times 25\% = 45\% + 5\% = 50\%$ ✓

### 1.3 解题四步法

间隔增长率问题看似复杂，但只要按照"四步走"的标准流程，就能做到快、准、稳：

#### 第一步：定时间
- **现期**：材料给出的较晚时间点
- **中间年份**：现期的前一年
- **间隔基期**：现期的前两年

**识别技巧**：看到"2023年...相比2021年增长"这样的表述，立即确定：
- 现期：2023年
- 中间年份：2022年  
- 间隔基期：2021年

#### 第二步：求$r_1$（中间年份的增长率）
材料通常不直接给出，需要通过以下线索推断：
- <span style={{color:'orange'}}>**"增速回落X个百分点"**</span>：用现期增长率减去回落数
- <span style={{color:'orange'}}>**"增速提高X个百分点"**</span>：用现期增长率减去提高数  
- <span style={{color:'orange'}}>**直接给出历年增长率**</span>：直接查找对应年份

#### 第三步：算R（间隔增长率）
代入公式：$R = r_1 + r_2 + r_1 \times r_2$

#### 第四步：得基期（间隔基期量）
使用公式：间隔基期量 = $\frac{现期量}{1 + R}$

### 1.4 三大速算技巧

| 技巧类型 | 适用场景 | 操作示例 | 精度评估 |
|---------|---------|---------|---------|
| **乘法估算** | $r_1$和$r_2$均&lt;10% | $6.1\% × 5.2\% \approx 0.3\%$ | 高精度 |
| **忽略乘积项** | $r_1$和$r_2$均&lt;5%且同向变化 | $R \approx r_1 + r_2$ | 中等精度 |
| **变形公式** | 选项差距小时 | $间隔基期量=\frac{现期量}{(1+r_1)(1+r_2)}$ | 最高精度 |

**详细说明**：

#### 技巧1：乘法估算法
当两个增长率都小于10%时，乘积项 $r_1 \times r_2$ 相对较小，可以进行快速估算：
- $8\% × 6\% = 0.48\% \approx 0.5\%$
- $12\% × 15\% = 1.8\% \approx 2\%$

#### 技巧2：忽略乘积项法  
当增长率都很小（&lt;5%）且方向相同时，乘积项几乎可以忽略：
- $3\% + 4\% + 3\% × 4\% = 7\% + 0.12\% \approx 7\%$

#### 技巧3：变形公式法
当精度要求很高时，直接使用变形：
$$间隔基期量 = \frac{现期量}{(1+r_1)(1+r_2)}$$

### 1.5 高频易错陷阱

#### 陷阱1：时间错位
<span style={{color:'orange'}}>**易错点**</span>：混淆"同比"（与上年同期比）和"环比"（与相邻周期比）

**正确理解**：
- "2023年增长8%"通常指与2022年相比
- "2023年一季度同比增长8%"指与2022年一季度相比

#### 陷阱2：单位陷阱  
<span style={{color:'orange'}}>**易错点**</span>：亿元与万亿元混用（1万亿元=10000亿元）

**规避方法**：计算前统一单位，或在最终答案中注意单位换算

#### 陷阱3：负增长处理
<span style={{color:'orange'}}>**易错点**</span>：当$r_1$或$r_2$为负时，$r_1 × r_2$结果为负值

**正确处理**：
- 正负×正负=正正  
- 正负×负正=负负
- 负负×负负=正正

#### 陷阱4：表述误读
<span style={{color:'orange'}}>**易错点**</span>："提高X个百分点"需减法计算

**示例**：增速10%，提高2个百分点 → 上年增速8%

## 二、真题精讲

### 主题1：基础间隔增长率计算

**例1**：
2023年，全国粮食总产量69541万吨，比上年增长1.3%。已知2022年全国粮食总产量比2021年增长0.5%。问2021年全国粮食总产量约为多少万吨？

A. 67821 <br />
B. 68456 <br /> 
C. 69106 <br />
D. 69541 <br />

<BlurredAnswer>
**第一步：定时间**
- 现期：2023年
- 中间年份：2022年
- 间隔基期：2021年

**第二步：求$r_1$（2022年相对2021年的增长率）**
- 题目直接给出：$r_1 = 0.5\%$

**第三步：求$r_2$（2023年相对2022年的增长率）**
- 题目直接给出：$r_2 = 1.3\%$

**第四步：计算间隔增长率R**
$$R = r_1 + r_2 + r_1 \times r_2$$
$$R = 0.5\% + 1.3\% + 0.5\% \times 1.3\%$$
$$R = 1.8\% + 0.0065\%$$
$$R = 1.8065\% \approx 1.8\%$$

<span style={{color:'red'}}>**速算技巧**</span>：由于$r_1$和$r_2$都很小，乘积项$0.5\% \times 1.3\% = 0.0065\%$可以忽略。

**第五步：计算间隔基期量**

$$ \text{2021年产量} = \frac{69541}{1 + 1.8\%} = \frac{69541}{1.018} \approx 68316 $$

使用化除为乘技巧：

$$ \frac{69541}{1.018} \approx 69541 \times (1 - 1.8\%) = 69541 - 69541 \times 1.8\% $$
$$ 69541 \times 1.8\% \approx 69541 \times 0.018 \approx 70000 \times 0.018 = 1260 $$
$$ \text{2021年产量} \approx 69541 - 1260 = 68281 $$

对比选项，B选项68456最接近。

**答案：B**
</BlurredAnswer>

### 主题2：增速回落类问题

**例2**：
2023年，某市规模以上工业增加值同比增长6.8%，增速比上年回落1.2个百分点。问与2021年相比，2023年该市规模以上工业增加值约增长了多少？

A. 14.9% <br />
B. 15.8% <br />
C. 16.4% <br />
D. 17.1% <br />

<BlurredAnswer>
**第一步：定时间**
- 现期：2023年
- 中间年份：2022年 
- 间隔基期：2021年

**第二步：求$r_1$（2022年增长率）**
- 2023年增长率：6.8%
- "增速比上年回落1.2个百分点"意思是2023年比2022年增速慢了1.2个百分点
- 因此：$r_1$ = 6.8% + 1.2% = 8.0%

<span style={{color:'red'}}>**关键理解**</span>："回落"意味着现期增速比上年低，所以上年增速 = 现期增速 + 回落数值

**第三步：求$r_2$（2023年增长率）**
- $r_2 = 6.8\%$

**第四步：计算间隔增长率R**
$$R = r_1 + r_2 + r_1 \times r_2$$
$$R = 8.0\% + 6.8\% + 8.0\% \times 6.8\%$$
$$R = 14.8\% + 0.544\%$$
$$R = 15.344\% \approx 15.3\%$$

**速算验证**：
$$8.0\% \times 6.8\% = 0.08 \times 0.068 = 0.00544 = 0.544\%$$

对比选项，B选项15.8%最接近。

**答案：B**
</BlurredAnswer>

### 主题3：增速提高类问题

**例3**：
2023年，全国网上零售额154264亿元，同比增长11.0%，增速比上年提高2.2个百分点。问2021年全国网上零售额约为多少亿元？

A. 125400 <br />
B. 128600 <br />
C. 131200 <br />
D. 134800 <br />

<BlurredAnswer>
**第一步：定时间**
- 现期：2023年
- 中间年份：2022年
- 间隔基期：2021年

**第二步：求$r_1$（2022年增长率）**
- 2023年增长率：11.0%
- "增速比上年提高2.2个百分点"意思是2023年比2022年增速快了2.2个百分点
- 因此：$r_1$ = 11.0% - 2.2% = 8.8%

<span style={{color:'red'}}>**关键理解**</span>："提高"意味着现期增速比上年高，所以上年增速 = 现期增速 - 提高数值

**第三步：求$r_2$（2023年增长率）**
- $r_2 = 11.0\%$

**第四步：计算间隔增长率R**
$$R = r_1 + r_2 + r_1 \times r_2$$
$$R = 8.8\% + 11.0\% + 8.8\% \times 11.0\%$$

乘积项计算：
$$8.8\% \times 11.0\% = 0.088 \times 0.11 = 0.00968 \approx 0.97\%$$

$$R = 19.8\% + 0.97\% = 20.77\% \approx 20.8\%$$

**第五步：计算间隔基期量**

$$ \text{2021年网上零售额} = \frac{154264}{1 + 20.8\%} = \frac{154264}{1.208} $$

使用首数法：

$$ \frac{154}{1.208} \approx \frac{154}{1.2} \approx 128.3 $$

因此约为128300亿元，对比选项，B选项128600最接近。

**答案：B**
</BlurredAnswer>

### 主题4：负增长混合问题

**例4**：
2023年，某省进出口总值8420亿元，同比增长15.2%。2022年该省进出口总值同比下降3.8%。问2021年该省进出口总值约为多少亿元？

A. 7456 <br />
B. 7628 <br />
C. 7891 <br />
D. 8133 <br />

<BlurredAnswer>
**第一步：定时间**
- 现期：2023年
- 中间年份：2022年
- 间隔基期：2021年

**第二步：识别增长率**
- $r_1 = -3.8\%$（2022年相对2021年，注意是负增长）
- $r_2 = 15.2\%$（2023年相对2022年）

**第三步：计算间隔增长率R**
$$R = r_1 + r_2 + r_1 \times r_2$$
$$R = (-3.8\%) + 15.2\% + (-3.8\%) \times 15.2\%$$
$$R = 11.4\% + (-0.5776\%)$$
$$R = 11.4\% - 0.58\% = 10.82\%$$

<span style={{color:'red'}}>**负增长乘积处理**</span>：
- 负数×正数=负数
- $(-3.8\%) \times 15.2\% = -0.5776\%$

**第四步：计算间隔基期量**

$$ \text{2021年进出口总值} = \frac{8420}{1 + 10.82\%} = \frac{8420}{1.1082} $$

$$ \approx \frac{8420}{1.1} \approx 7654 $$

对比选项，B选项7628比较接近。

**精确计算验证**：

$$ \frac{8420}{1.1082} \approx 7597 $$

仍然是B选项最接近。

**答案：B**
</BlurredAnswer>

### 主题5：复杂表述解读问题

**例5**：
2023年，全国固定资产投资503036亿元，同比增长3.0%。其中，基础设施投资增长5.9%，制造业投资增长6.5%，房地产开发投资下降9.6%。已知2022年全国固定资产投资同比增长5.1%。问2021年全国固定资产投资约为多少万亿元？

A. 46.8 <br />
B. 47.5 <br />
C. 48.2 <br />
D. 49.1 <br />

<BlurredAnswer>
**第一步：定时间和主体**
- 现期：2023年
- 中间年份：2022年
- 间隔基期：2021年
- 主体：全国固定资产投资（不是分项投资）

**第二步：提取关键数据**
- 现期量：503036亿元
- $r_1 = 5.1\%$（2022年增长率）
- $r_2 = 3.0\%$（2023年增长率）

<span style={{color:'orange'}}>**陷阱警示**</span>：题目给了很多分项投资的增长率，但问的是总体投资，不要被干扰信息误导。

**第三步：计算间隔增长率R**
$$R = r_1 + r_2 + r_1 \times r_2$$
$$R = 5.1\% + 3.0\% + 5.1\% \times 3.0\%$$
$$R = 8.1\% + 0.153\%$$
$$R = 8.253\% \approx 8.3\%$$

**第四步：计算间隔基期量**

$$ \text{2021年投资} = \frac{503036}{1 + 8.3\%} = \frac{503036}{1.083} $$

单位换算：503036亿元 = 50.3036万亿元

$$ \frac{50.3036}{1.083} \approx 46.45 \text{万亿元} $$

对比选项，A选项46.8最接近。

**答案：A**
</BlurredAnswer>

### 主题6：间隔基期比重计算

**例6**：
2023年，某地第三产业增加值1840亿元，占GDP的比重为58.2%，同比增长7.8%。同期，该地GDP同比增长6.5%。已知2022年第三产业增加值同比增长5.2%，GDP同比增长4.8%。问2021年第三产业增加值占GDP的比重约为多少？

A. 55.9% <br />
B. 56.8% <br />
C. 57.4% <br />
D. 58.2% <br />

<BlurredAnswer>
**第一步：理解题目类型**
这是间隔基期比重问题，需要求2021年（间隔基期）的比重。

**第二步：收集比重相关数据**
- 2023年比重：58.2%
- 第三产业增长率：$a_1 = 5.2\%$（2022年），$a_2 = 7.8\%$（2023年）
- GDP增长率：$b_1 = 4.8\%$（2022年），$b_2 = 6.5\%$（2023年）

**第三步：使用间隔基期比重公式**
$$间隔基期比重 = \frac{现期比重}{部分间隔增长系数} \times 整体间隔增长系数$$

其中：
- 部分间隔增长系数 = $(1 + a_1) \times (1 + a_2) = 1.052 \times 1.078 = 1.134$
- 整体间隔增长系数 = $(1 + b_1) \times (1 + b_2) = 1.048 \times 1.065 = 1.116$

**第四步：计算**
$$间隔基期比重 = \frac{58.2\%}{1.134} \times 1.116$$
$$= 58.2\% \times \frac{1.116}{1.134}$$
$$= 58.2\% \times 0.984$$
$$= 57.3\%$$

对比选项，C选项57.4%最接近。

**答案：C**
</BlurredAnswer>

### 主题7：间隔基期平均数计算

**例7**：
2023年，某市规模以上工业企业1256家，实现主营业务收入15680亿元，同比增长12.8%。企业数量同比增长3.2%。已知2022年主营业务收入同比增长8.5%，企业数量同比增长1.8%。问2021年该市规模以上工业企业户均主营业务收入约为多少亿元？

A. 10.8 <br />
B. 11.2 <br />
C. 11.6 <br />
D. 12.1 <br />

<BlurredAnswer>
**第一步：理解题目**
- 户均收入 = 总收入 / 企业数量
- 求2021年（间隔基期）的户均收入

**第二步：计算2023年（现期）户均收入**

$$ \text{2023年户均收入} = \frac{15680}{1256} = 12.48 \text{亿元} $$

**第三步：计算间隔增长率**
主营业务收入的间隔增长率：
$$R_收入 = 8.5\% + 12.8\% + 8.5\% \times 12.8\% = 21.3\% + 1.088\% = 22.388\%$$

企业数量的间隔增长率：
$$R_数量 = 1.8\% + 3.2\% + 1.8\% \times 3.2\% = 5.0\% + 0.0576\% = 5.0576\%$$

**第四步：使用间隔基期平均数公式**

$$ \text{间隔基期平均数} = \frac{\text{现期平均数}}{(1 + R_\text{总量})} \times (1 + R_\text{数量}) $$

$$ \text{2021年户均收入} = \frac{12.48}{1 + 22.388\%} \times (1 + 5.0576\%) $$
$$ = \frac{12.48}{1.22388} \times 1.050576 $$
$$ = 10.198 \times 1.050576 $$
$$ = 10.715 \text{亿元} $$

对比选项，A选项10.8最接近。

**答案：A**
</BlurredAnswer>

### 主题8：间隔基期同比增长率

**例8**：
2023年，全国社会消费品零售总额471495亿元，同比增长7.2%。已知2022年同比增长12.6%，2021年同比增长12.5%。问2023年社会消费品零售总额比2021年增长了多少？

A. 19.8% <br />
B. 20.1% <br />
C. 20.7% <br />
D. 21.3% <br />

<BlurredAnswer>
**第一步：理解题目**
- 现期：2023年
- 间隔基期：2021年
- 问题：求2023年相对于2021年的增长率（即间隔增长率R）。

**第二步：识别增长率**
- 2023年相对2022年的增长率 $r_2$：题目给出"同比增长7.2%"，所以 $r_2 = 7.2\%$。
- 2022年相对2021年的增长率 $r_1$：题目给出"同比增长12.6%"，所以 $r_1 = 12.6\%$。
- <span style={{color:'orange'}}>**干扰信息**</span>：题目中"2021年同比增长12.5%"是为了计算2021年相对2020年的增长率，与本题无关，是干扰项。

**第三步：计算间隔增长率**
根据间隔增长率公式：
