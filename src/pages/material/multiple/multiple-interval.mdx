# 间隔倍数

## 核心概念

### 什么是间隔倍数？

<span style={{color:'red'}}>间隔倍数是指现期数值与基期数值的比值，且两者之间相隔一个中间时期</span>。

让我们用一个生活中的例子来理解：

想象你在关注某个网红的粉丝数变化：
- 2022年：100万粉丝（基期）
- 2023年：增长了20%，达到120万粉丝（中间期）
- 2024年：在2023年基础上又增长了30%，达到156万粉丝（现期）

那么2024年的粉丝数是2022年的多少倍？这就是间隔倍数问题。

### 核心公式推导

设：
- 基期量为 $A$
- 中间期增长率为 $r_1$
- 现期增长率为 $r_2$（注意：是现期相对于中间期的增长率）

**第一步：**理解增长过程

$$中间期量 = A \times (1 + r_1)$$

$$现期量 = 中间期量 \times (1 + r_2) = A \times (1 + r_1) \times (1 + r_2)$$

**第二步：**推导间隔倍数
$$间隔倍数 = \frac{现期量}{基期量} = \frac{A × (1 + r_1) × (1 + r_2)}{A} = (1 + r_1) × (1 + r_2)$$

**第三步：**展开公式
$$(1 + r_1) × (1 + r_2) = 1 + r_1 + r_2 + r_1 × r_2$$

设<span style={{color:'red'}}>间隔增长率 $R = r_1 + r_2 + r_1 × r_2$</span>，则：

$$间隔倍数 = 1 + R$$

### 公式记忆技巧

<span style={{color:'green'}}>记忆口诀：两率相加再相乘，结果加一是倍数</span>

公式简化：当 $r_1 × r_2$ 的值很小时（通常小于1%），可以忽略不计：
$$间隔倍数 ≈ 1 + r_1 + r_2$$

## 真题讲解

### 主题1：企业营收增长问题

**例1（2020年国考）**：
某企业2018年营业收入为500万元，2017年同比增长10%，2018年同比增长20%。问：2018年营业收入是2016年的多少倍？

<BlurredAnswer>
**解题步骤：**

**第一步：**识别时间轴和各期数据
- 基期：2016年（设为 A）
- 中间期：2017年，增长率 $r_1 = 10\%$
- 现期：2018年，增长率 $r_2 = 20\%$

**第二步：**计算间隔增长率
$$R = r_1 + r_2 + r_1 × r_2 = 10\% + 20\% + 10\% × 20\% = 30\% + 2\% = 32\%$$

**第三步：**计算间隔倍数
$$间隔倍数 = 1 + R = 1 + 32\% = 1.32$$

**答案：**2018年营业收入是2016年的1.32倍。

**验证：**$(1 + 10\%) × (1 + 20\%) = 1.1 × 1.2 = 1.32$ ✓
</BlurredAnswer>

### 主题2：人口增长问题

**例2**：
某市2023年人口为80万人，2022年人口增长率为5%，2023年人口增长率为8%。求2023年人口是2021年人口的多少倍？

<BlurredAnswer>
**解题步骤：**

**第一步：**明确时间关系
- 基期：2021年
- 中间期：2022年，增长率 $r_1 = 5\%$
- 现期：2023年，增长率 $r_2 = 8\%$

**第二步：**应用间隔倍数公式
$$间隔倍数 = (1 + r_1) × (1 + r_2) = (1 + 5\%) × (1 + 8\%) = 1.05 × 1.08 = 1.134$$

**答案：**2023年人口是2021年人口的1.134倍。

**思路总结：**直接套用公式即可，无需计算具体数值。
</BlurredAnswer>

### 主题3：GDP增长问题

**例3**：
某地区2024年GDP为1200亿元，2023年GDP增长率为6%，2024年GDP增长率为7%。问：2024年GDP相比2022年增长了多少倍？

<BlurredAnswer>
**解题步骤：**

**第一步：**<span style={{color:'orange'}}>注意题目陷阱</span>
题目问的是"增长了多少倍"，不是"是多少倍"，需要先求倍数再减1。

**第二步：**计算间隔倍数
$$间隔倍数 = (1 + 6\%) × (1 + 7\%) = 1.06 × 1.07 = 1.1342$$

**第三步：**计算增长倍数
$$增长倍数 = 间隔倍数 - 1 = 1.1342 - 1 = 0.1342$$

**答案：**2024年GDP相比2022年增长了0.1342倍。
</BlurredAnswer>

### 主题4：股票投资问题

**例4**：
某股票2023年价格为50元，2022年涨幅为15%，2023年涨幅为-10%（下跌10%）。问：2023年股价是2021年股价的多少倍？

<BlurredAnswer>
**解题步骤：**

**第一步：**识别增长率（含负增长）
- $r_1 = 15\%$
- $r_2 = -10\%$

**第二步：**计算间隔倍数
$$间隔倍数 = (1 + 15\%) × (1 + (-10\%)) = 1.15 × 0.9 = 1.035$$

**答案：**2023年股价是2021年股价的1.035倍。

**知识点：**负增长率的处理方法，直接代入公式即可。
</BlurredAnswer>

### 主题5：销售额波动问题

**例5**：
某商店2024年销售额为300万元，2023年销售额增长率为25%，2024年销售额增长率为-20%。问：2024年销售额是2022年销售额的多少倍？

A. 0.8倍 <br />
B. 1.0倍 <br />
C. 1.2倍 <br />
D. 1.5倍 <br />

<BlurredAnswer>
**解题步骤：**

**第一步：**识别各期增长率
- $r_1 = 25\%$
- $r_2 = -20\%$

**第二步：**计算间隔倍数
$$间隔倍数 = (1 + 25\%) × (1 + (-20\%)) = 1.25 × 0.8 = 1.0$$

**答案：**B. 1.0倍

**启发：**经过一升一降，销售额又回到了原来的水平。
</BlurredAnswer>

### 主题6：复杂计算问题

**例6**：
某企业2024年利润为800万元，2023年利润增长率为12%，2024年利润增长率为15%。已知2022年利润为x万元，求x的值。

<BlurredAnswer>
**解题步骤：**

**第一步：**建立等式关系
$$现期量 = 基期量 × 间隔倍数$$
$$800 = x × (1 + 12\%) × (1 + 15\%)$$

**第二步：**计算间隔倍数
$$间隔倍数 = 1.12 × 1.15 = 1.288$$

**第三步：**求解x
$$x = \frac{800}{1.288} = 621.12$$

**答案：**2022年利润为621.12万元。
</BlurredAnswer>

## 技巧总结

### 解题三步法

1. **第一步：**<span style={{color:'red'}}>识别时间轴</span>
   - 基期、中间期、现期
   - 各期对应的增长率

2. **第二步：**<span style={{color:'red'}}>套用公式</span>
   - 间隔倍数 = $(1 + r_1) × (1 + r_2)$
   - 或间隔倍数 = $1 + r_1 + r_2 + r_1 × r_2$

3. **第三步：**<span style={{color:'red'}}>注意区分</span>
   - "是多少倍"：直接用倍数
   - "增长了多少倍"：倍数减1

### 常见易错点

1. **时间陷阱**：<span style={{color:'orange'}}>注意题目问的是哪一年相对于哪一年</span>
2. **增长率陷阱**：<span style={{color:'orange'}}>分清是环比增长率还是同比增长率</span>
3. **表述陷阱**：<span style={{color:'orange'}}>区分"是多少倍"和"增长了多少倍"</span>
4. **负增长处理**：<span style={{color:'orange'}}>负增长率直接代入公式，不需要特殊处理</span>

### 快速估算技巧

当 $r_1 × r_2$ 很小时（通常小于1%），可以使用简化公式：
$$间隔倍数 ≈ 1 + r_1 + r_2$$

例如：$r_1 = 3\%$，$r_2 = 4\%$
- 精确计算：$(1 + 3\%) × (1 + 4\%) = 1.0712$
- 近似计算：$1 + 3\% + 4\% = 1.07$
- 误差很小，可以接受
