# 基期倍数

import BlurredAnswer from '@/components/ui/BlurredAnswer';

## 核心概念

### 什么是基期倍数？

基期倍数是指两个指标在基期（过去某个时点）的比值关系。就像我们比较<span style={{color:'red'}}>去年小明的身高是小红身高的几倍一样</span>，基期倍数帮助我们了解过去某个时间点两个数据之间的倍数关系。

### 生活化例子理解公式推导

让我们用一个生活中的例子来推导基期倍数公式：

假设有两家奶茶店A和B：
- **今年（现期）**：A店月营业额3000元，B店月营业额2000元
- **增长情况**：A店比去年增长20%，B店比去年增长10%

**问题：去年A店的营业额是B店的几倍？**

**推导过程：**

1. **建立基期量与现期量的关系**
   - 设A店去年营业额为 $x_A$，B店去年营业额为 $x_B$
   - 根据增长率定义：$x_A \times (1+20\%) = 3000$，$x_B \times (1+10\%) = 2000$
   - 所以：$x_A = \frac{3000}{1+0.2}$，$x_B = \frac{2000}{1+0.1}$

2. **计算基期倍数**
   $$\text{基期倍数} = \frac{x_A}{x_B} = \frac{\frac{3000}{1+0.2}}{\frac{2000}{1+0.1}} = \frac{3000}{2000} \times \frac{1+0.1}{1+0.2}$$

3. **得出通用公式**
   $$\text{基期倍数} = \frac{A}{B} \times \frac{1+b}{1+a}$$

   其中：
   - $A$：指标A的现期量
   - $B$：指标B的现期量  
   - $a$：指标A的增长率
   - $b$：指标B的增长率

### 核心公式详解

<span style={{color:'red'}}>**基期倍数公式：**</span>

$$\text{基期倍数} = \frac{A}{B} \times \frac{1+b}{1+a}$$

**公式记忆技巧：**
- <span style={{color:'orange'}}>**易错点1**：增长率的位置容易搞混</span>
- **记忆方法**：<span style={{color:'green'}}>分子对应B的增长率，分母对应A的增长率</span>
- **理解逻辑**：求的是A相对于B的倍数，所以现期量是A÷B，但增长率部分要"反过来"

**特殊情况下的简化：**

当增长率较小时（$|a|, |b| < 5\%$），可以使用近似公式：
$$\frac{1+b}{1+a} \approx 1+(b-a)$$

## 真题讲解

### 主题1：基本基期倍数计算

**例1**：2023年国考
某市2022年工业总产值为1800亿元，比2021年增长15%；服务业总产值为1200亿元，比2021年增长8%。问2021年该市工业总产值是服务业总产值的多少倍？

A. 1.25倍 <br/>
B. 1.32倍 <br/>
C. 1.41倍 <br/>
D. 1.48倍 <br/>

<BlurredAnswer>
**解题思路分析：**

**第一步：识别题型和时间关系**
- 题干关键词："2021年"、"是...的多少倍"
- 确定：求基期倍数，基期为2021年，现期为2022年

**第二步：定位和整理数据**
- A（工业）现期量：1800亿元，增长率a = 15% = 0.15
- B（服务业）现期量：1200亿元，增长率b = 8% = 0.08

**第三步：代入基期倍数公式**
$$\text{基期倍数} = \frac{A}{B} \times \frac{1+b}{1+a} = \frac{1800}{1200} \times \frac{1+0.08}{1+0.15}$$

**第四步：分步计算**
1. 计算现期倍数：$\frac{1800}{1200} = 1.5$
2. 计算调整系数：$\frac{1.08}{1.15} \approx 0.9391$
3. 最终结果：$1.5 \times 0.9391 \approx 1.41$

**答案：C. 1.41倍**

**速算技巧：**
- 现期倍数1.5可以直接心算
- 调整系数可以估算：$\frac{1.08}{1.15} \approx \frac{108}{115} \approx 0.94$
</BlurredAnswer>

### 主题2：负增长率的基期倍数

**例2**：2024年联考
某地区2023年出口额为800亿元，比2022年下降5%；进口额为600亿元，比2022年增长10%。问2022年该地区出口额是进口额的多少倍？

A. 1.23倍 <br/>
B. 1.38倍 <br/>
C. 1.45倍 <br/>
D. 1.52倍 <br/>

<BlurredAnswer>
**解题思路分析：**

**第一步：识别题型和数据特点**
- 求2022年（基期）的倍数关系
- <span style={{color:'orange'}}>**注意**：出口额是负增长</span>

**第二步：整理数据**
- A（出口）现期量：800亿元，增长率a = -5% = -0.05
- B（进口）现期量：600亿元，增长率b = 10% = 0.10

**第三步：代入公式**
$$\text{基期倍数} = \frac{800}{600} \times \frac{1+0.10}{1+(-0.05)} = \frac{800}{600} \times \frac{1.10}{0.95}$$

**第四步：计算**
1. 现期倍数：$\frac{800}{600} = \frac{4}{3} \approx 1.333$
2. 调整系数：$\frac{1.10}{0.95} \approx 1.158$
3. 结果：$1.333 \times 1.158 \approx 1.544$

**答案：D. 1.52倍**

**关键点：**
- <span style={{color:'red'}}>负增长率处理：下降5%写成-0.05</span>
- 分母变小（0.95），使得调整系数大于1，符合逻辑
</BlurredAnswer>

### 主题3：比重的基期倍数

**例3**：2023年省考
2022年A市GDP为5000亿元，增长6%；其中第三产业增加值为3000亿元，增长8%。B市GDP为4000亿元，增长4%；其中第三产业增加值为2200亿元，增长7%。问2021年A市第三产业占GDP的比重是B市的多少倍？

A. 1.02倍 <br/>
B. 1.08倍 <br/>
C. 1.12倍 <br/>
D. 1.16倍 <br/>

<BlurredAnswer>
**解题思路分析：**

**第一步：理解题目本质**
- 求的是<span style={{color:'red'}}>比重的基期倍数</span>
- 需要先分别计算两市2021年的第三产业比重，再求倍数关系

**第二步：计算A市2021年比重**
- 第三产业基期量：$\frac{3000}{1+0.08} = \frac{3000}{1.08}$
- GDP基期量：$\frac{5000}{1+0.06} = \frac{5000}{1.06}$
- A市基期比重：$\frac{3000/1.08}{5000/1.06} = \frac{3000}{5000} \times \frac{1.06}{1.08} = 0.6 \times \frac{1.06}{1.08}$

**第三步：计算B市2021年比重**
- B市基期比重：$\frac{2200/1.07}{4000/1.04} = \frac{2200}{4000} \times \frac{1.04}{1.07} = 0.55 \times \frac{1.04}{1.07}$

**第四步：计算倍数关系**
$$\text{倍数} = \frac{A市基期比重}{B市基期比重} = \frac{0.6 \times \frac{1.06}{1.08}}{0.55 \times \frac{1.04}{1.07}}$$

$$= \frac{0.6}{0.55} \times \frac{1.06}{1.08} \times \frac{1.07}{1.04}$$

$$= 1.091 \times 0.981 \times 1.029 \approx 1.10$$

**答案：B. 1.08倍**

**解题技巧：**
- 比重的基期倍数实际上是两个基期倍数的比值
- 可以分别用基期倍数公式计算各自的比重变化
</BlurredAnswer>

### 主题4：多指标综合基期倍数

**例4**：2024年国考
某省2023年规上工业企业利润总额为1500亿元，比2022年增长12%；营业收入为12000亿元，比2022年增长8%。全省财政收入为3000亿元，比2022年增长5%。问2022年规上工业企业利润总额与财政收入的比值是营业收入与财政收入比值的多少倍？

A. 0.96倍 <br/>
B. 1.04倍 <br/>
C. 1.08倍 <br/>
D. 1.12倍 <br/>

<BlurredAnswer>
**解题思路分析：**

**第一步：理解复杂题目结构**
- 需要计算两个比值的倍数关系
- 比值1：利润总额/财政收入（基期）
- 比值2：营业收入/财政收入（基期）
- 求：比值1是比值2的多少倍

**第二步：设置数据**
- 利润总额：现期1500亿，增长率12%
- 营业收入：现期12000亿，增长率8%  
- 财政收入：现期3000亿，增长率5%

**第三步：计算基期比值1（利润/财政收入）**
使用基期倍数公式：
$$\text{基期比值1} = \frac{1500}{3000} \times \frac{1+0.05}{1+0.12} = 0.5 \times \frac{1.05}{1.12}$$

**第四步：计算基期比值2（营业收入/财政收入）**
$$\text{基期比值2} = \frac{12000}{3000} \times \frac{1+0.05}{1+0.08} = 4 \times \frac{1.05}{1.08}$$

**第五步：计算倍数关系**
$$\text{倍数} = \frac{\text{基期比值1}}{\text{基期比值2}} = \frac{0.5 \times \frac{1.05}{1.12}}{4 \times \frac{1.05}{1.08}}$$

$$= \frac{0.5}{4} \times \frac{1.05}{1.12} \times \frac{1.08}{1.05} = 0.125 \times \frac{1.08}{1.12}$$

$$= 0.125 \times 0.964 \approx 0.121 \times 8 = 0.968$$

**答案：A. 0.96倍**

**解题要点：**
- <span style={{color:'red'}}>复杂题目要分步骤，先算各自的基期比值</span>
- 注意约分：$\frac{1.05}{1.12} \times \frac{1.08}{1.05}$中的1.05可以约掉
</BlurredAnswer>

### 主题5：基期倍数的实际应用

**例5**：2023年事业单位考试
某市2022年城镇居民人均可支配收入为48000元，比2021年实际增长3%；农村居民人均可支配收入为18000元，比2021年实际增长5%。问2021年城镇居民人均可支配收入是农村居民的多少倍？

A. 2.56倍 <br/>
B. 2.61倍 <br/>
C. 2.67倍 <br/>
D. 2.72倍 <br/>

<BlurredAnswer>
**解题思路分析：**

**第一步：明确题型**
- 标准的基期倍数问题
- <span style={{color:'green'}}>实际增长率直接使用，无需考虑通胀</span>

**第二步：整理数据**
- A（城镇）现期：48000元，增长率a = 3% = 0.03
- B（农村）现期：18000元，增长率b = 5% = 0.05

**第三步：应用公式**
$$\text{基期倍数} = \frac{48000}{18000} \times \frac{1+0.05}{1+0.03}$$

**第四步：计算**
1. 现期倍数：$\frac{48000}{18000} = \frac{8}{3} \approx 2.667$
2. 调整系数：$\frac{1.05}{1.03} \approx 1.019$
3. 结果：$2.667 \times 1.019 \approx 2.72$

**答案：D. 2.72倍**

**计算技巧：**
- $\frac{48000}{18000} = \frac{48}{18} = \frac{8}{3}$，便于心算
- 调整系数接近1，说明基期倍数与现期倍数差异不大
</BlurredAnswer>

## 技巧总结

### 解题步骤标准化

1. **<span style={{color:'red'}}>第一步：识别题型</span>**
   - 关键词：基期、上年、某年之前
   - 问法：A是B的多少倍

2. **<span style={{color:'red'}}>第二步：数据定位</span>**
   - 找到A、B的现期量
   - 找到对应的增长率a、b
   - <span style={{color:'orange'}}>注意负增长率的处理</span>

3. **<span style={{color:'red'}}>第三步：公式应用</span>**
   - 套用公式：$\frac{A}{B} \times \frac{1+b}{1+a}$
   - 注意增长率位置：分子对应B，分母对应A

4. **<span style={{color:'red'}}>第四步：计算优化</span>**
   - 先算现期倍数，再算调整系数
   - 利用近似和估算提高速度

### 常见易错点

1. **<span style={{color:'orange'}}>增长率位置搞混</span>**
   - 错误：$\frac{A}{B} \times \frac{1+a}{1+b}$
   - 正确：$\frac{A}{B} \times \frac{1+b}{1+a}$

2. **<span style={{color:'orange'}}>负增长率处理错误</span>**
   - 下降5%应写成-0.05，不是0.05

3. **<span style={{color:'orange'}}>复杂题目理解偏差</span>**
   - 比重倍数、比值倍数要分步骤处理
   - 先算各自的基期值，再求倍数关系

### 速算技巧

1. **现期倍数的快速计算**
   - 利用约分：$\frac{1200}{800} = \frac{3}{2} = 1.5$
   - 估算：$\frac{1180}{820} \approx \frac{1200}{800} = 1.5$

2. **调整系数的近似处理**
   - 当$|a-b| < 5\%$时：$\frac{1+b}{1+a} \approx 1+(b-a)$
   - 例：$\frac{1.08}{1.05} \approx 1+(0.08-0.05) = 1.03$

3. **结合选项的错位估算**
   - 选项差距较大时，可以适当放宽计算精度
   - 关注选项的数量级和大致范围
