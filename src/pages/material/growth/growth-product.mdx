# 乘积增长率

import BlurredAnswer from '@/components/ui/BlurredAnswer';

## 核心概念

### 什么是乘积增长率？

乘积增长率是指当一个量等于两个或多个量的乘积时，如何通过各个分量的增长率来计算总量的增长率。这在资料分析中是一个非常重要的概念。

让我们用一个<span style={{color:'red'}}>生活中的例子</span>来理解：

假设小明开了一家奶茶店，他的**总收入 = 奶茶单价 × 销售杯数**

- 2023年：奶茶单价15元/杯，销售了10000杯，总收入 = 15 × 10000 = 150000元
- 2024年：由于原料涨价，奶茶单价涨到16.5元/杯（增长10%），但由于竞争激烈，销售量只有9500杯（下降5%）

问题：小明2024年的总收入增长率是多少？

### 公式推导过程

设总量 $A = B \times C$，其中：
- $A$ 是总量（如总收入）
- $B$ 是第一个分量（如单价）  
- $C$ 是第二个分量（如销量）

<span style={{color:'red'}}>第一步：建立变化后的关系式</span>

初始状态：$A_0 = B_0 \times C_0$

变化后状态：$A_1 = B_1 \times C_1 = B_0(1+r_B) \times C_0(1+r_C)$

其中 $r_B$、$r_C$ 分别是 $B$、$C$ 的增长率。

<span style={{color:'red'}}>第二步：展开乘积</span>

$$A_1 = B_0(1+r_B) \times C_0(1+r_C) = B_0C_0(1+r_B)(1+r_C)$$

$$A_1 = A_0(1+r_B)(1+r_C) = A_0(1 + r_B + r_C + r_Br_C)$$

<span style={{color:'red'}}>第三步：计算总增长率</span>

$$r_A = \frac{A_1 - A_0}{A_0} = \frac{A_0(1 + r_B + r_C + r_Br_C) - A_0}{A_0}$$

$$r_A = r_B + r_C + r_Br_C$$

这就是<span style={{color:'red'}}>乘积增长率的精确公式</span>！

### 近似公式的应用

当 $r_B$ 和 $r_C$ 都比较小（通常小于10%）时，交叉项 $r_Br_C$ 是一个很小的数，可以忽略：

$$r_A \approx r_B + r_C$$

回到小明奶茶店的例子：
- 单价增长率：$r_B = 10\% = 0.1$
- 销量增长率：$r_C = -5\% = -0.05$

**精确计算**：$r_A = 0.1 + (-0.05) + 0.1 \times (-0.05) = 0.05 - 0.005 = 0.045 = 4.5\%$

**近似计算**：$r_A \approx 0.1 + (-0.05) = 0.05 = 5\%$

验证：2024年总收入 = 16.5 × 9500 = 156750元，增长率 = (156750-150000)/150000 = 4.5%

可见精确公式更准确！

### <span style={{color:'orange'}}>易错点提醒</span>

1. **符号问题**：负增长率要带负号计算
2. **单位问题**：确保所有量的单位一致
3. **交叉项**：当增长率较大时，不能忽略 $r_Br_C$ 项
4. **适用范围**：只适用于乘积关系，不适用于加减关系

## 真题讲解

### 主题一：价格销量类问题

**例1**（模拟2024年国考）：
某超市2023年苹果总销售额为120万元。2024年苹果价格上涨8%，销售量下降3%，则2024年苹果总销售额的增长率约为：

A. 5.0% <br />
B. 4.8% <br />
C. 4.6% <br />
D. 5.2% <br />

<BlurredAnswer>
**解题思路**：

**第一步**：识别乘积关系
总销售额 = 价格 × 销售量

**第二步**：确定各分量增长率
- 价格增长率：$r_1 = 8\% = 0.08$
- 销售量增长率：$r_2 = -3\% = -0.03$

**第三步**：应用乘积增长率公式
由于增长率都小于10%，可以使用近似公式：
$r_{总} \approx r_1 + r_2 = 8\% + (-3\%) = 5\%$

**第四步**：验证精确计算
$r_{总} = r_1 + r_2 + r_1r_2 = 0.08 + (-0.03) + 0.08 \times (-0.03) = 0.05 - 0.0024 = 0.0476 = 4.76\%$

**答案**：B. 4.8%（最接近精确值）
</BlurredAnswer>

**例2**（模拟2023年省考）：
某电商平台2022年某商品订单总金额为500万元，2023年该商品平均订单金额增长12%，订单数量增长15%，则2023年订单总金额的增长率为：

A. 27.0% <br />
B. 28.8% <br />
C. 26.5% <br />
D. 29.2% <br />

<BlurredAnswer>
**解题思路**：

**第一步**：识别关系
订单总金额 = 平均订单金额 × 订单数量

**第二步**：确定增长率
- 平均订单金额增长率：$r_1 = 12\% = 0.12$
- 订单数量增长率：$r_2 = 15\% = 0.15$

**第三步**：由于增长率较大，使用精确公式
$r_{总} = r_1 + r_2 + r_1r_2 = 0.12 + 0.15 + 0.12 \times 0.15 = 0.27 + 0.018 = 0.288 = 28.8\%$

**第四步**：对比近似计算
近似：$r_{总} \approx 0.12 + 0.15 = 27\%$，误差达1.8%，说明精确公式的重要性。

**答案**：B. 28.8%
</BlurredAnswer>

### 主题二：人口经济类问题

**例3**（模拟2025年国考）：
某地区2023年GDP为2000亿元，2024年人口增长0.5%，人均GDP增长6.2%，则该地区2024年GDP增长率为：

A. 6.7% <br />
B. 6.73% <br />
C. 6.8% <br />
D. 7.0% <br />

<BlurredAnswer>
**解题思路**：

**第一步**：识别关系
GDP = 人均GDP × 人口

**第二步**：确定增长率
- 人均GDP增长率：$r_1 = 6.2\% = 0.062$
- 人口增长率：$r_2 = 0.5\% = 0.005$

**第三步**：应用公式（增长率较小，可用近似）
$r_{GDP} \approx r_1 + r_2 = 6.2\% + 0.5\% = 6.7\%$

**第四步**：精确验证
$r_{GDP} = 0.062 + 0.005 + 0.062 \times 0.005 = 0.067 + 0.00031 = 0.06731 = 6.731\%$

四舍五入约为6.73%

**答案**：B. 6.73%
</BlurredAnswer>

### 主题三：农业生产类问题

**例4**（模拟2024年省考）：
某农场2023年小麦总产量为800吨，2024年种植面积扩大5%，亩产量提高8%，则2024年小麦总产量增长率为：

A. 13.0% <br />
B. 13.4% <br />
C. 13.8% <br />
D. 12.6% <br />

<BlurredAnswer>
**解题思路**：

**第一步**：识别关系
总产量 = 种植面积 × 亩产量

**第二步**：确定增长率
- 种植面积增长率：$r_1 = 5\% = 0.05$
- 亩产量增长率：$r_2 = 8\% = 0.08$

**第三步**：应用精确公式
$r_{总产量} = r_1 + r_2 + r_1r_2 = 0.05 + 0.08 + 0.05 \times 0.08 = 0.13 + 0.004 = 0.134 = 13.4\%$

**第四步**：验证
2024年总产量 = 800 × (1+0.05) × (1+0.08) = 800 × 1.05 × 1.08 = 907.2吨
增长率 = (907.2-800)/800 = 13.4%

**答案**：B. 13.4%
</BlurredAnswer>

### 主题四：工业生产类问题

**例5**（模拟2023年联考）：
某工厂2022年产品总价值为1500万元，2023年产品数量减少4%，单位产品价值增长9%，则2023年产品总价值的增长率为：

A. 4.6% <br />
B. 5.0% <br />
C. 4.2% <br />
D. 4.8% <br />

<BlurredAnswer>
**解题思路**：

**第一步**：识别关系
产品总价值 = 产品数量 × 单位产品价值

**第二步**：确定增长率
- 产品数量增长率：$r_1 = -4\% = -0.04$
- 单位产品价值增长率：$r_2 = 9\% = 0.09$

**第三步**：应用精确公式
$r_{总价值} = r_1 + r_2 + r_1r_2 = (-0.04) + 0.09 + (-0.04) \times 0.09 = 0.05 - 0.0036 = 0.0464 = 4.64\%$

约为4.6%

**第四步**：<span style={{color:'orange'}}>注意负增长率的处理</span>
交叉项为负数，使得总增长率小于两个分量增长率的简单相加。

**答案**：A. 4.6%
</BlurredAnswer>

### 主题五：复杂多因子问题

**例6**（模拟2025年省考）：
某企业利润 = 营业收入 × 利润率，2024年营业收入增长15%，利润率从8%提高到9%，则2024年利润增长率为：

A. 28.1% <br />
B. 28.8% <br />
C. 29.4% <br />
D. 27.5% <br />

<BlurredAnswer>
**解题思路**：

**第一步**：识别关系
利润 = 营业收入 × 利润率

**第二步**：计算利润率增长率
利润率从8%提高到9%，增长率 = (9%-8%)/8% = 1%/8% = 12.5%

**第三步**：确定各增长率
- 营业收入增长率：$r_1 = 15\% = 0.15$
- 利润率增长率：$r_2 = 12.5\% = 0.125$

**第四步**：应用精确公式
$r_{利润} = r_1 + r_2 + r_1r_2 = 0.15 + 0.125 + 0.15 \times 0.125 = 0.275 + 0.01875 = 0.29375 = 29.375\%$

约为29.4%

**答案**：C. 29.4%
</BlurredAnswer>

## 技巧总结

### <span style={{color:'red'}}>核心公式记忆</span>

1. **精确公式**：$r_A = r_B + r_C + r_Br_C$
2. **近似公式**：$r_A \approx r_B + r_C$（当$|r_B|, |r_C| < 10\%$时）

### <span style={{color:'red'}}>快速判断技巧</span>

1. **识别乘积关系**的关键词：
   - 总收入、总销售额、总产值
   - 总产量、总人口、GDP
   - 一般形式：总量 = 分量1 × 分量2

2. **选择公式的判断标准**：
   - 增长率绝对值都 < 10%：用近似公式
   - 有增长率绝对值 ≥ 10%：用精确公式
   - 考试时间紧：优先近似公式，再检验

3. **<span style={{color:'orange'}}>易错陷阱</span>**：
   - 混淆加法和乘法关系
   - 忘记交叉项的影响
   - 负增长率符号处理错误
   - 百分比和小数转换错误

### <span style={{color:'red'}}>解题步骤模板</span>

1. **识别关系**：确认是否为 $A = B \times C$ 的形式
2. **提取数据**：找出各分量的增长率
3. **选择公式**：根据增长率大小选择精确或近似公式
4. **计算求解**：代入公式计算
5. **验证检查**：检查答案合理性

### <span style={{color:'green'}}>时间节省技巧</span>

- 当交叉项 $r_Br_C$ 很小时（如两个5%的乘积只有0.25%），直接忽略
- 记住常见百分比的小数形式：5%=0.05，12%=0.12等
- 学会快速心算：如15%×8%=1.2%

### <span style={{color:'purple'}}>拓展应用</span>

乘积增长率不仅适用于两个因子，也适用于多个因子：
- 三因子：$A = B \times C \times D$时，$r_A \approx r_B + r_C + r_D$
- 实际应用：总产值 = 企业数 × 平均规模 × 单位产值
