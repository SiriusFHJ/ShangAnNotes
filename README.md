## 📖 项目简介

ShangAnNotes 是一个专注于**中国公务员考试**的备考资料库，涵盖了**行测**和**申论**的所有核心知识点。本项目由具有10年以上公务员考试教学经验的专家团队精心编写，旨在帮助考生系统化掌握考试内容，提高应试能力。

### ✨ 特色亮点

- 📚 **系统全面**：覆盖行测5大模块和申论全部题型
- 🎯 **实战导向**：结合历年真题，注重解题技巧
- 💡 **易学易懂**：使用通俗语言，配有详细例题解析
- 🔄 **持续更新**：紧跟考试变化，及时更新内容
- 📱 **多种格式**：支持 Markdown 和 MDX 格式，便于阅读

## 🎯 学习建议

### 📅 备考时间线

- **基础阶段**（1-2个月）：系统学习各模块基础知识
- **强化阶段**（1-2个月）：大量练习，掌握解题技巧
- **冲刺阶段**（2-4周）：模拟考试，查漏补缺

### 📖 学习方法

1. **理论先行**：先理解概念和原理
2. **大量练习**：通过练习巩固知识点
3. **总结归纳**：定期整理错题和心得
4. **模拟实战**：严格按照考试时间练习

### 💡 重点提示

- 数量关系可适当放弃，但基础题型要掌握
- 言语理解重在语感培养和技巧运用
- 判断推理注重逻辑思维训练
- 资料分析是提分关键，必须重点突破
